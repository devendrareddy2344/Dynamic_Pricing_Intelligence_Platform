# Frontend (Live Observability Dashboard)

Serves as the primary Operations dashboard (React / Vite) styled deeply with Tailwind CSS plugins.

**Responsibilities:**
1. Subscribes dynamically to `EventSource` backend channels streaming from the API Gateway.
2. Visualizes real-time performance of the target scrapers (Hit/Fail mapping).
3. Interprets GenAI Markdown elements utilizing `remark-gfm` extensions to inject structural, enterprise-grade styled tables directly onto the dashboard for competitive positioning views.
